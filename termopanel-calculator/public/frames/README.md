# Обрамления окон (фото-референс)

Сюда положите фото установленного обрамления окон (наличник с пилястрами/карнизом).
Имя файла = `id` записи из `lib/frames.ts`:

`/frames/{id}.jpg`

| id (в каталоге) | Файл | Название |
|---|---|---|
| `frame1` | `frame1.jpg` | Классика |
| `frame2` | `frame2.jpg` | Классика 2 |

Если файла нет — приложение не падает: в сетке показывается заглушка-квадрат,
а Gemini работает по текстовому описанию `hint` (без референс-картинки).
