# Кронштейны (фото-референс)

Сюда положите фото кронштейна/консоли. Имя файла = `id` записи из `lib/brackets.ts`:

`/brackets/{id}.jpg`

| id (в каталоге) | Файл | Название · размер |
|---|---|---|
| `bracket1` | `bracket1.jpg` | КА-31 · 43×14 см |
| `bracket2` | `bracket2.jpg` | КА-32 · 30×18 см |
| `bracket3` | `bracket3.jpg` | КА-33 · 26.5×15 см |

Если файла нет — приложение не падает: показывается заглушка-квадрат, а Gemini
работает по текстовому описанию `hint`.
