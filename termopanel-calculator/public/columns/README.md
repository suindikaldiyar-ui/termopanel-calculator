# Угловые колонны (фото-референс)

Сюда положите фото угловой колонны/пилястры (закрывает угол здания).
Имя файла = `id` записи из `lib/columns.ts`:

`/columns/{id}.jpg`

| id (в каталоге) | Файл | Название |
|---|---|---|
| `column1` | `column1.jpg` | Классика |

Если файла нет — приложение не падает: в сетке показывается заглушка-квадрат,
а Gemini работает по текстовому описанию `hint` (без референс-картинки).
