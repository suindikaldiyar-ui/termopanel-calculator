# Текстуры цоколя / фундамента

Сюда положите фото отделки цоколя (close-up / бесшовные плитки).
Имя файла должно совпадать с `id` записи из `lib/foundations.ts`:

`/foundations/{id}.jpg`

Пример (когда добавите запись в `FOUNDATIONS`):

| id (в каталоге) | Файл |
|---|---|
| `graphite` | `graphite.jpg` |
| `grey` | `grey.jpg` |
| `beige` | `beige.jpg` |

Если файла нет — приложение не падает: в сетке показывается цветной `swatch`
(fallback), а Gemini использует текстовое описание `hint`.
