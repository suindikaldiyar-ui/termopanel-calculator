# Термопанельные планки (фото-референс)

Планки, которые ставятся вокруг окон. Имя файла = `id` записи из `lib/termopanels.ts`:

`/termopanels/{id}.jpg`

| id (в каталоге) | Файл | Название · размер |
|---|---|---|
| `tp25` | `tp25.jpg` | КА-25 · 30×3 см |
| `tp26` | `tp26.jpg` | КА-26 · 40×3 см |

Если файла нет — приложение не падает: показывается заглушка-квадрат, а Gemini
работает по текстовому описанию `hint`.
