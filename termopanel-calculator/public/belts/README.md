# Межэтажные пояса (фото-референс)

Сюда положите фото профиля межэтажного пояса (горизонтальный декор между этажами).
Имя файла = `id` записи из `lib/belts.ts`:

`/belts/{id}.jpg`

| id (в каталоге) | Файл | Название · размер |
|---|---|---|
| `belt1` | `belt1.jpg` | КА-22 · 20×7 см |
| `belt2` | `belt2.jpg` | КА-22 · 25×8 см |
| `belt3` | `belt3.jpg` | КА-22 · 30×10 см |
| `belt4` | `belt4.jpg` | КА-23 · 24.5×7 см |
| `belt5` | `belt5.jpg` | КА-24 · 29.5×14 см |

Если файла нет — приложение не падает: в сетке показывается заглушка-квадрат,
а Gemini работает по текстовому описанию `hint` (без референс-картинки).
