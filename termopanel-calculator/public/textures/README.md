# Текстуры травертина

Сюда положите реальные фото текстур (close-up / бесшовные плитки).
Имена файлов должны совпадать с `id` цвета из `lib/textures.ts`:

| Файл | Цвет |
|---|---|
| `klassik.jpg` | Классик |
| `krem.jpg` | Крем |
| `kofe.jpg` | Кофе |
| `kapuchino.jpg` | Капучино |
| `seriy.jpg` | Серый |
| `pesok.jpg` | Песок |

Если файла нет — приложение не падает:
- в каталоге показывается цветной swatch (fallback);
- Gemini генерирует по текстовому описанию (`hint`), без reference-фото.
